utility module

utility.check_types(*upper_args, **upper_kwargs)

Decorator checking all the function’s arguments’ types to match the signature. For in-class methods first argument given to this decorator must be Any.

Taken from https://ru.stackoverflow.com/

Parameters:

  • upper_args – Types for the respective positional function’s arguments

  • upper_kwargs – Types for the respective keyword function’s arguments

Raises:

TypeError – If the arguments’ types doesn’t match the signature
