benchmark module

class benchmark.BenchMarker(tree_heights: range, number: int, repeat: int)

Bases: object

A class created to collect all the variables needed to benchmark different tree-making algorithms and to plot the respective graph in a single place.

tree_heights

Heights of trees for test data.

Type:

range

res_recursive_dict

Dict with time amounts needed to execute recursive tree-making algorithm saving the tree in a dict.

Type:

list[float]

res_iterative_dict

Dict with time amounts needed to execute bfs tree-making algorithm saving the tree in a dict.

Type:

list[float]

res_recursive_nodes

Dict with time amounts needed to execute recursive tree-making algorithm saving the tree in a nodes.

Type:

list[float]

res_iterative_nodes

Dict with time amounts needed to execute bfs tree-making algorithm saving the tree in a nodes.

Type:

list[float]

number

The amount of times the method timeit will be called for each tree-making algorithm. It’s set to 100 by default.

Type:

int

repeat

The amount of times the method timeit will execute the “func” for each tree-making algorithm. It’s set to 5 by default.

Type:

int

benchmarking_foreach()

Estimates different tree-making algorithms’ speed and saves grades to its args.

create_graphs()

Creates the graph of tree-making algorithms’ speed and Saves it to ‘../img/outputImage.png’.
