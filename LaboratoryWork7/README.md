# Lab's files structure:
- `src/currency.py` - currencies api
- `src/decorators.py` - logging and checking decorators
- `src/file_logging.py` - an example of using `logging.Logger` class in combine with a logging decorator
- `src/demonstration.py` - code presenting the equations solver being decorated
- `src/logs.txt` - logs created by `file_logging.py`
- `src/logs2.txt` - logs created by `demonstration.py`
- `test_currency.py` - the file with tests
